Overview#

FracAbility is a Python toolbox that can be used to analyse fracture networks in digitalized rock outcrops.

These are the main features available :

  • Shapefile importing support

  • Rapid topology analysis and identification of I,Y,X and U nodes

  • Fracture network backbone(s) identification

  • Statistical analysis tools:

    • Empirical CDF and SF calculation

    • Distribution fitting

    • Statistical model testing

  • Plotting tools:

    • Network objects plotting using matplotlib or vtk

    • Ternary node plot

    • Rose diagram

    • Statistical plotting

The name FracAbility recalls the reliability library that inspired and helped in the creation of this project.
